# Archive — snapshots de design historiques

Ces fichiers `DESIGN_vX.md` sont des **instantanés de conception** pris au fil des versions
(v0.18 → v0.41). Ils décrivent l'intention d'une version donnée **au moment où elle a été
écrite** et ne sont **plus maintenus**.

> ⚠️ Ne pas les traiter comme la source de vérité. L'état **réel** du jeu vit dans le code
> (`src/`) et dans la documentation vivante : [`../ARCHITECTURE.md`](../ARCHITECTURE.md),
> [`../DESIGN.md`](../DESIGN.md) et [`../systemes/`](../systemes/).

Ils sont conservés ici pour la **traçabilité** des décisions (pourquoi telle mécanique
existe, ce qui a été essayé puis abandonné). Pour l'historique complet, voir aussi
`git log`.
